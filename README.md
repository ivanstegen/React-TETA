# React-School

A collection of React learning projects and demos.

## Projects

### 1. Children Prop Demo
A demonstration of using the children prop in React components.

### 2. Component Tree Demo
Examples of various React components including:
- Tenzies game
- Login forms
- Locker box component
- And more...

## Getting Started

Each project folder contains its own `package.json`. To run a project:

1. Navigate to the project folder
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
